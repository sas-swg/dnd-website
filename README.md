# dnd website

D&D 5e Guide / DnD 5e Guide
Online: https://sas-swg.github.io/dnd-website/

An unofficial D&D 5th Edition guide in Ukrainian. Includes character classes, races, backstories, spells, feats, rules, and a character generator with PDF export. Can be used entirely offline.

Classes, races, backstories, spells, feats, and rules. Character generator with PDF export (supports Cyrillic, also works offline). Quick search function. Responsive website for phones and tablets. Offline access after download.
